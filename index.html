Setting up ExpressJS server

  1. mkdir waffle-fries
  2. cd waffle-fries
  3. npm init (press enter a million times)
  4. npm install express --save
  5. Add additional script in package.json "start": "node index.js"
  6. Create file index.html
  7. Put something fun in your index.html file
  8. Create file index.js
  9. Paste the following into index.js
    var express = require('express'); var path = require('path'); var port = process.env.PORT || 3000; var app = express(); app.get('/', function(req, res) { res.sendFile(path.join(__dirname, './index.html')); }); app.listen(port, function(err) { if (err) { console.log(err); } });
  10. node index.js
  11. Go to http://localhost:3000
  12. See your page!

Deploying to Heroku

  1. Create a heroku account if you do not already have one at www.heroku.com
  2. Install heroku cli
  3. Create a new repo in git and push changes from Part 1
    • Go to github and create a new repository
    • Copy HTTPS clone url from github
    • git init
    • git remote add origin {HTTPS url}
    • git add .
    • git commit -m "First commit"
    • git push origin master
  4. heroku login (with your heroku credentials)
  5. heroku create
  6. git push heroku master
  7. heroku open
Note: when testing locally, your change should be reflected at http://localhost:3000
When you want to deploy any changes to heroku, run git push origin master AND git push heroku master. Your change should then be reflected at your heroku site
